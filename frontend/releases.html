Upcoming Release Dates

Release Date page is currently down due to API architecture changes.

How it works (Phase 2): The goal here is to completely automate the weekly Wednesday pull list. The legacy version relied on simple web scraping, but I'm currently re-architecting it to be a more resilient asynchronous event bus. The backend uses Java to pull structured release data (eventually hooking into the League of Comic Geeks API), formats it, and pushes those dates directly into the user's Google Calendar. I'm focusing heavily on fault tolerance right now so that a single misaligned JSON response doesn't crash the entire scheduling pipeline.
