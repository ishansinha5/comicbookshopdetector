Recommender Page

Recommender page is currently down due to API architecture changes.

How it works (Phase 2): I am currently rebuilding the ML pipeline from the ground up. The original iteration was a collaborative filtering model trained on a massive dataset I scraped from Goodreads and Amazon. The new architecture is built around compute optimization. I'm deploying a PyTorch vision model wrapped in a Python FastAPI endpoint. To keep things 'Green' and avoid burning through GPU cycles, I engineered a native C++ perceptual hashing (pHash) layer that acts as a bouncer. When you upload a comic cover, C++ hashes it instantly at the byte level and checks the PostgreSQL cache. If it's a known image, it skips the heavy Python inference entirely. Honestly, it's slightly overkill for a simple comic app, but it's a great sandbox for learning enterprise-grade efficiency.
