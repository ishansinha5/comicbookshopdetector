About This Project

The Mission: Skipping the big-box retailers. This distributed microservices platform connects readers with local comic shops, tracks weekly release schedules, and offers smart recommendations. Built to explore compute-optimized "Green AI" and spatial routing architecture—and to scientifically ensure I never miss a pull day.

The Architecture: Under the hood, this bypasses the standard, bloated API calls. It runs on a containerized WSL ecosystem, utilizing a native C++ Perceptual Hashing (pHash) layer as a "bouncer." By checking byte-level hashes against a PostGIS spatial database, it aggressively caches image data before it ever hits the Python ML model. It's designed to save massive amounts of expensive GPU cycles while maintaining lightning-fast frontend lookups.

The Developer: Built for the community, by an engineer who is slightly obsessed with efficient backend infrastructure.
