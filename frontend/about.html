About This Project

The What: The Comic Shop Curato is a project designed for comic book enthusiasts. It helps you discover local comic book shops, get personalized comic book recommendations, and stay up-to-date with new release schedules.

The How: This was built using HTML, CSS, Python and JavaScript, leveraging various APIs to provide real-time data for shop locations and recommendations.

The Why: The comic book industry thrives off of the local comic book stores, not doing business with big retailers. This project is intended to be a full stack solution to that - supporting local businesses and new media alike.

The Where: This website will work everywhere, but unfortunately only for those using Chrome.

The Who: This project was created for all comic book enthusiasts, by a comic book enthusiast with a passion for programming.
