💥 Comic Shop Curator 🗯

Shop Finder is currently down due to API architecture changes.

How it works (Phase 2): Right now, this page is offline while I transition the backend from a serverless monolith to a localized microservices stack. Previously, I was pulling location data through the Google Places API, which worked, but felt a bit bloated for what I needed. Now, I'm routing client requests through a Java (Spring Boot) gateway straight into a PostGIS spatial database running inside WSL. By seeding the database with local shop coordinates, I can use pure mathematical spatial indexing to calculate distances rather than relying on an expensive third-party API. It's a lot of plumbing to set up manually, but the lookup speeds are significantly faster.
